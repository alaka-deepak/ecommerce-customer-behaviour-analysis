## Retail Customer Behaviour & Profitability Optimisation

### Business Problem
Understanding customer shopping behaviour to improve personalisation, conversion rates, and overall retail profitability in an eCommerce environment.

### Dataset
Public UK online retail transaction dataset containing customer purchases, product details, pricing, and timestamps.

### Approach
- Data cleaning and transformation using Python and SQL-style analysis
- Exploratory analysis of customer purchasing patterns
- Customer segmentation using RFM metrics and clustering
- Predictive modelling to estimate purchase propensity
- A/B testing framework design to evaluate model impact

### Tools & Technologies
Python, SQL, Pandas, NumPy, Scikit-learn, XGBoost, Power BI / Tableau

### Business Impact
Insights from this project support:
- Targeted customer personalisation
- Improved customer retention strategies
- Data-driven commercial decision-making
